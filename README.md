# ZNC - 基于Webpack2.0的多页应用架构

## How to use?

````
# Clone this repo https://github.com/weiTimes/Webpack2.0-Staging.git

# Install dependencies
  npm install || yarn(If you have installed)

# Run for development
  npm run start

# Run for production
  npm run build
````
